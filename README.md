# Python and TensorFlow Programming Course

This repository contains the code and notes from the "Python and TensorFlow Programming" course.

## Getting Started

1.  **Environment Check**: Python is already installed on this system.
2.  **Dependencies**: You will need to install TensorFlow. Run the following command:
    ```bash
    pip install tensorflow
    ```
3.  **Course File**: `course_notebook.py` is initialized and ready for your code.

## Git Workflow

To push your progress:
```bash
git add .
git commit -m "Add new course work"
git push origin master
```
*(Make sure to add your remote repository URL using `git remote add origin <URL>`)*
